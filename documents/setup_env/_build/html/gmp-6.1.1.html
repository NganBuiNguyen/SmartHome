Download gmp-6.1.1.tar.gz and moving to folder thirparties.

The step install are:

Step 1: tar -xvf gmp-6.1.1.tar.gz

Step 2: cd gmp-6.1.1

Step 3: ./configure –prefix=/development/gmp-6.1.1

Step 4: make

Step 5: make install
