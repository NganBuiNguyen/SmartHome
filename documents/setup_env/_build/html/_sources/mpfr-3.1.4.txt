III. Setting mpfr-3.1.4:
========================

Download mpfr-3.1.4.tar.gz and moving to folder thirparties.

**The step install are:**

Step 1: tar -xvf mpfr-3.1.4.tar.gz //extract 

Step 2: cd mpfr-3.1.4

Step 3: ./configure --prefix=/development/mpfr-3.1.4

Step 4: make

Step 5: make install
