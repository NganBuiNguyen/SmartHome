IV. Setting mpc-1.0.2
======================

Download mpc-1.0.2.tar.gz and moving to folder thirparties.

**The step install are:**

Step 1: tar -xvf mpc-1.0.2.tar.gz //extract 

Step 2: cd mpc-1.0.2

Step 3: ./configure --with-mpfr=/development/mpfr-3.1.4 --prefix=/development/mpc-1.0.2

Step 4: make 

Step 5: make install
